# 2-Simple-Music-Player

The application is a music player with a stylish GUI. Users can register or log in, create playlists, and play pre-stored songs with standard playback controls.

## Features
- User registration and login system
- Playlist creation by adding song names
- Music playback with controls:
  - Play / Pause
  - Next / Previous song
  - Repeat functionality
- GUI interface for easy interaction

## How to Run
1. Download or clone the repository.
2. Open the `2SimpleMusicPlayer2`.
3. Navigate to `src/com/simplemusicplayer2/`.
4. Run the `Main.java` file.
5. The login/register page will appear:
   - Register with a username and password.
   - Use those credentials to log in.
6. Go to the Playlist section:
   - Type the name of a pre-stored song.
   - Click on "ADD" to open the player.
7. Use the playback controls (keys/buttons) to play, pause, repeat, or switch songs.

## Notes
- Developed and tested in IntelliJ IDEA.
- Songs are pre-stored within the project.
